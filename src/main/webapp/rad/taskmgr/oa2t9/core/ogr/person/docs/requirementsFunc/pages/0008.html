1. 在职人员：当点击一下在职人员这一标签时，它会把在职人员以树状列表的形式展现出来，刚开始只显示最顶层，其它隐藏。当点击前面的+号，它会把下一级列出来，此层为第二级。当点击第二级时，它会把此层下的所有人员在右边都列出来，可以对人员进行增加、编辑、删除。也可以点击左边树状结构的最末级对人员进行编辑。
  2. 离职人员/外部人员：当点击一下离职人员/外部人员这一标签时，它会把当前所有的离职人员/外部人员在右边都列出来，可以对人员进行增加、编辑、删除。
  3. 用户查询和导出：当点击一下用户查询和导出这一标签时，右边会出现查询用户这一页面，当用户输入数据时，它会把查询到的结果输出到当前页面中。
  4. 用户导入：当点击一下用户导入这一标签时，右边会出现导入用户这一页面，点击浏览按钮，选择需要导入的csv文件后，点击导入按钮
  5. 批量个性用户设置：当点击一下批量个性用户设置这一标签时，右边会出现批量个性用户设置这一页面，当点击批量设置这一按钮时，会提交成功。
